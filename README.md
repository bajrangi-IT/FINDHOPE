FindHope — Missing People Finder

FindHope is a simple and helpful web app made to report, search, and track missing people in one place. In cities like Delhi-NCR, missing cases are increasing every year, and information is often scattered. FindHope brings all reports together so people, police, and NGOs can respond faster.

Features

Add a Missing Person report with photo and last-seen details

Add a Found Person / sighting report

Search cases quickly by name or location

Simple, mobile-friendly UI for all users

Optional AI-based matching to suggest possible matches

Easy sharing on social media

Tech Stack

HTML, CSS, JavaScript (frontend),
Node/Express or Firebase (backend),
MongoDB/Firestore for database

Goal

To reduce search time, increase awareness, and help more missing people return home safely.
